## 关键学习心得

1. **先跑通，再优化** — 每个版本只解决一类核心问题
2. **抽象分层** — 接口层 → 服务层 → 数据库层；页面 → 组件 → 工具函数
3. **重复代码是坏味道** — 看到重复就抽离组件/Hook/工具函数
4. **状态管理是前端核心难点** — 从页面自管 → 刷新信号 → 最终需要状态管理库
5. **文档是最好的学习助手** — 写计划明确方向，写总结巩固记忆，写问题避免再踩坑
