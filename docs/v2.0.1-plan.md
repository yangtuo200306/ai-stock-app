## 后端改动（3 个文件）

### 1. 新增 `backend/app/services/market_service.py`

大盘行情 + 板块排行数据获取，参考 `news_fetcher.py` 的模式（独立模块 + 内存缓存 + 异常处理）。

**结构：**

```python

    _sector_cache: dict   # TTL 300 秒

        # 调 ak.stock_zh_index_spot_sina()
        # 过滤出主要指数：上证指数、深证成指、创业板指、科创50、沪深300
        # 返回 {indices: [{name, code, price, change_pct, change_amount}], summary: "..."}

    def get_sector_rankings(top_n: int = 5) -> dict
        # 调 ak.stock_board_industry_name_em()
        # 取涨幅前 top_n + 跌幅前 top_n
        # 返回 {top: [{name, code, change_pct}], bottom: [...], summary: "..."}
```

**关键设计：**
- 缓存：大盘 120 秒，板块 300 秒（内存 dict + 时间戳）
- 错误处理：akshare 异常时返回 `{"error": "数据源暂时不可用"}`，不抛异常
- 返回结构包含 `summary` 字段，方便 LLM 直接引用

### 2. 修改 `backend/app/services/tool_factory.py`

注册 2 个新工具：

```python
# get_market_indices
ToolDefinition(
    name="get_market_indices",
    description="获取A股主要指数实时行情，包括上证指数、深证成指、创业板指等。当用户询问大盘走势、市场整体表现时调用。",
    parameters=[],
    handler=_handle_get_market_indices,
)

# get_sector_rankings
ToolDefinition(
    name="get_sector_rankings",
    description="获取行业板块涨跌排行。当用户询问哪些板块涨得好、板块轮动情况时调用。",
    parameters=[
        ToolParameter(name="top_n", type="integer", description="返回前N个板块，默认5", required=False),
    ],
    handler=_handle_get_sector_rankings,
)
```

**关键设计：**
- 描述包含触发条件，帮助 LLM 判断什么时候调
- `get_market_indices` 无参数，降低 LLM 调用门槛
- `get_sector_rankings` 的 top_n 默认 5，手机屏幕有限

## 前端改动（4 个文件）

### 1. 修改 `mobile-app/src/navigation/AppNavigator.tsx`

"问股"tab 指向新增的 DrawerNavigator：

```typescript
// 当前
<Tab.Screen name="问股" component={AskScreen} />

// 改为
<Tab.Screen name="问股" component={AskDrawerNavigator} />
```

新增 `AskDrawerNavigator`：

```typescript
import { createDrawerNavigator } from '@react-navigation/drawer';

const Drawer = createDrawerNavigator();

function AskDrawerNavigator() {
  return (
    <Drawer.Navigator
      drawerContent={(props) => <SessionDrawer {...props} />}
      screenOptions={{
        headerShown: false,
        drawerType: 'front',
        drawerStyle: { width: 280 },
      }}
    >
      <Drawer.Screen name="AskMain" component={AskScreen} />
    </Drawer.Navigator>
  );
}
```

**关键设计：**
- `drawerType: 'front'` — 抽屉覆盖在 AskScreen 上面，不影响布局
- `drawerStyle: { width: 280 }` — 手机屏幕宽度有限，280px 够用

### 2. 新增 `mobile-app/src/components/SessionDrawer.tsx`

左侧抽屉内容组件：

```typescript
// 顶部：标题"历史会话" + 关闭按钮
// 列表：FlatList 显示 sessions
// 每条：title + 相对时间 + 消息数
// 点击：获取该 session 的消息 → restoreSession → 关闭抽屉
// 底部：无会话时的空状态提示
```

**关键设计：**
- 点击会话 → 调 `GET /api/sessions/{id}/messages` → `restoreSession(id, messages)` → 关闭抽屉
- 显示相对时间（"3 分钟前"、"昨天"）
- 无 stock_code 的 session 显示问题前 20 字作为标题
