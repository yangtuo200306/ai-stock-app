# AI Stock App v1.0 版本计划：前端架构升级（Zustand 状态管理）

## 1. 版本定位

v1.0 定位为：

```text
前端架构升级 — 引入 Zustand 替代 DataRefreshContext 版本号信号模式
```

v0.9 用 DataRefreshContext 解决了"页面间需要刷新同步"的问题，但本质是补丁方案。
v1.0 引入 Zustand，用真正的共享状态替代版本号信号模式，为后续页面精修打好地基。

## 2. 版本目标

- 引入 Zustand 状态管理库
- 创建 3 个 store：watchlistStore、recordsStore、askStore
- 迁移 4 个页面：WatchlistScreen、RecordListScreen、AskScreen、TaskStatusScreen
- 删除 DataRefreshContext 及其所有引用
- API 层增强：401 自动清 token
- 类型按领域拆分（5 个文件）
- 新增 resetAllStores 集中重置机制（登出/401 时一键清空）

## 3. 保留不变

- AuthContext（认证状态保持 Context 模式）
- 所有 UI 样式
- 后端代码
- 功能范围
