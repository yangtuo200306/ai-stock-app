## 实施检查清单

### 1. 密码可见切换（LoginScreen）— ✅ 已完成
- **文件**: `mobile-app/src/screens/LoginScreen.tsx`
- **改动**: 密码框和确认密码框右侧添加眼睛图标按钮，点击切换 `secureTextEntry`
- **后端**: 无改动
- **复杂度**: ⭐ 极小

### 2. 评分仪表盘（ScoreGauge 组件）— ✅ 已在 v1.5 前完成
- **新建组件**: `mobile-app/src/components/ScoreGauge.tsx`
  - 水平彩色进度条，纯 RN View 实现
  - 颜色分级: 0-40 红 / 40-70 黄 / 70-100 绿
  - 右侧显示分数数字
- **详情页替换**: 
  - `RecordDetailScreen.tsx` — 替换纯文本评分为 ScoreGauge
  - `ReportDetailScreen.tsx` — 替换纯文本评分为 ScoreGauge
- **列表页颜色标签**:
  - `AskScreen.tsx` — 评分数字加颜色
  - `RecordCard.tsx` — 评分数字加颜色
- **后端**: 无改动
- **复杂度**: ⭐⭐ 较小

### 3. 自选卡片重构（WatchlistStockCard + WatchlistScreen + 后端 + 导航）— ✅ 已在 v1.2 完成
- **后端**: `backend/app/api/stocks.py` 中 `GET /api/stocks` 返回时附带最新行情（price + change_pct）
- **类型**: 更新 `mobile-app/src/types/stock.ts` 中 `Stock` 接口，增加 `price` 和 `change_pct` 可选字段
- **导航**: `WatchlistStackParamList` 增加 `RecordDetail: { recordId: number }` 路由
- **卡片改动**:
  - 保留: 代码、名称、上次记录摘要、类型色条/标签
  - 新增: 实时价格 + 涨跌幅（带涨跌色）
  - 新增: "分析"按钮（替代原来点击卡片创建分析任务）
  - 修改: 点击卡片 → 查看记录详情（跳转 RecordDetail）
  - 无记录时: 提示"暂无记录"
- **Bug 修复**: `askStore.ts` 中 `handleAsk()` 成功后调用 `useWatchlistStore.getState().loadStocks()` 刷新自选摘要
- **复杂度**: ⭐⭐⭐ 中等

### 4. 记录页删除功能（RecordListScreen + recordsStore + 后端）— ✅ 已在 v1.2~v1.3 完成
- **后端新增**: `backend/app/api/records.py` 新增 `DELETE /api/records/{record_id}`
  - 校验记录归属当前用户
  - 删除后返回成功
- **Store 新增**: `recordsStore.ts` 新增 `deleteRecord(id: number)` action
  - 调用 `apiDelete('/api/records/' + id)`
  - 成功后调用 `fetchRecords()` 刷新列表
- **前端改动**:
  - `RecordCard.tsx` 右下角增加删除按钮（🗑️ 图标）
  - 点击弹出确认弹窗（Alert）
  - 确认后调用 store 的 `deleteRecord`
  - 删除成功后刷新列表
- **复杂度**: ⭐⭐ 较小

### 5. 后端搜索兜底（stock_resolver）— ✅ 已在 v1.6 计划前完成
- **文件**: `backend/app/services/stock_resolver.py`
- **改动**: `resolve_stock_input()` 增加一步：当正则和内置名称映射都匹配不到时，用 `stocks.index.json` 做模糊名称搜索
- **前提**: 依赖第 6 项生成的 `stocks.index.json`
- **复杂度**: ⭐ 极小

### 6. 股票自动补全（后端索引 + 前端组件）— ✅ 已在 v1.6 计划前完成
- **后端**:
  - 预生成 `backend/data/stocks.index.json`（A 股代码+名称列表）
  - 启动时加载到内存
  - 新增 `GET /api/stocks/search?q=xxx` 接口，按代码前缀或名称模糊匹配，返回前 10 条
- **前端组件**: 新建 `mobile-app/src/components/StockAutocomplete.tsx`
  - 输入时 300ms 防抖调用搜索接口
  - 匹配到的股票以 Tag/Chip 形式展示在输入框上方，可点击 × 移除
  - 选中后自动填充股票代码到输入框
- **应用场景**:
  - `AskScreen.tsx` — 问股输入框
  - `WatchlistScreen.tsx` — 自选添加表单
- **复杂度**: ⭐⭐⭐⭐ 较大
