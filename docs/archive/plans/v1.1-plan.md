# AI Stock App v1.1 版本计划：问股页精修

## 1. 版本定位

```text
阶段二：页面精修 — 问股页从"功能验证型"升级为"产品体验型"
```

v1.0 完成了全局架构升级（前端 Zustand + 后端配置/日志/错误码/数据库迁移），地基已稳。
v1.1 开始进入阶段二：逐个页面精修，第一个目标是问股页。

## 2. 目标

- 布局从"卡片内瀑布流"改为"聊天式全屏布局"
- 消息气泡支持 Markdown 渲染和复制
- 空状态有引导文案和快捷问股按钮
- 结果卡片分组展示（行情概览 / 技术指标 / 风险提示）
- 错误展示升级为内联警示 + 重试按钮
- 输入区固定在底部，支持多行自动增高
- 消息更新自动滚动到底部

## 3. 范围

### 改的

- `mobile-app/src/screens/AskScreen.tsx` — 重写布局和交互
- `mobile-app/src/components/MessageBubble.tsx` — 重写（Markdown + 复制 + 样式）
- `mobile-app/src/components/MetricRow.tsx` — 增强（multiline + compact 模式）
- `mobile-app/package.json` — 新增依赖

### 不改

- 后端 API 不改一行
- 不新增后端接口
- 不新增页面 / 不修改导航结构
- 不引入新的大功能（会话恢复、股票自动补全等）
- 不改整体视觉风格（颜色/字体/间距体系保持不变）

## 6. 验证方式

- `npx tsc --noEmit` 编译零错误
- 核心链路验证：输入问题 → 发送 → 展示消息（Markdown 渲染） → 展示结果卡片 → 加入自选 → 新建会话
- 错误场景：空输入不发送、失败后重试、加入自选失败提示
- 交互体验：自动滚动、多行输入、键盘弹出/收回
