# v1.9.1 Prompt 工程优化计划

## 版本目标

提升 AI 回答质量，不改架构、不加页面、不改数据源。

## 改动范围

仅一个文件：`backend/app/services/llm_client.py`

## 具体改动

### 1. System Prompt 增强

**当前：**
```python
system_prompt = "你是股票分析助手"
```

**改后：**
```python
system_prompt = (
    "你是专业的股票分析助手，擅长用通俗易懂的语言向个人投资者解释股票走势。\n"
    "你的分析风格：客观、简洁、有依据。\n"
    "你输出的每个结论都必须基于给定的数据，不猜测、不编造。\n"
    "回答控制在 200-300 字之间，适合手机阅读。"
)
```

### 2. 回答结构强制要求

在 user prompt 末尾追加结构要求：

```python
structure_instruction = (
    "\n\n请按以下结构回答：\n\n"
    "【结论】一句话直接回答用户问题\n\n"
    "【关键数据】列出最重要的 3-4 个数据点，关键数字用 **加粗** 标注\n"
    "  - 例如：当前价：**1880.50**，涨跌幅：**+1.25%**\n\n"
    "【详细分析】展开说明技术指标的含义和对趋势的影响\n\n"
    "【风险提示】列出主要风险点"
)
```

### 3. 数据分类呈现

当前数据段平铺所有指标，改为按分类组装：

```python
# 改后数据段结构：
prompt_parts = [
    "## 行情概览",
    f"- 当前价：{quote['price']} | 涨跌幅：{quote['change_pct']}%",
    f"- 换手率：{quote['turnover_rate']}% | 振幅：{quote['amplitude']}%",
    "",
    "## 趋势指标",
    f"- MA5(5日均线): {indicators['ma5']} | MA10(10日均线): {indicators['ma10']} | MA20(20日均线): {indicators['ma20']}",
    f"- 均线趋势：{indicators['ma_trend']}",
    "",
    "## 动量指标",
    f"- RSI(6)(相对强弱指标): {indicators['rsi']}（{indicators['rsi_interpretation']}）",
    f"- 乖离率(MA5): {indicators['bias']}%",
    "",
    "## 成交量分析",
    f"- 成交量比值：{indicators['volume_ratio']}",
    f"- 成交量信号：{indicators['volume_signal']}",
]
```
