## 3. 已实现功能

### MineScreen 重构

**用户头像区：**
- 64px 圆形头像占位，主题色背景 + 白色首字母
- 下方显示用户名（已登录）或"未登录"
- 状态标签：绿色圆点 + "已登录" / 灰色圆点 + "未登录"

**账号操作区：**
- 紧凑 Pressable 按钮风格
- 已登录：红色背景"退出登录"
- 未登录：主题色背景"登录 / 注册"

**应用信息区：**
- 独立卡片，居中展示
- 应用名称 "AI Stock" + 版本号（从 `src/constants/app.ts` 读取，不再硬编码）
- 简短描述

**开发者设置（可折叠）：**
- 默认收起，标题行 `▶ 开发者设置`（展开后变为 `▼`）
- 展开后保持原有后端地址配置逻辑不变
- 保存地址 + 测试连接功能正常

### LoginScreen 视觉升级

- 输入框之间增加视觉分隔（`inputDivider`）
- 错误提示改为带红色背景的圆角 Badge 风格
- 切换模式按钮改为紧凑 Pressable 风格
- 文案从"没有账号？点击注册"改为"还没有账号？去注册"

### 版本常量文件

- 新增 `src/constants/app.ts`，集中管理 `APP_VERSION` 和 `APP_NAME`
- 一处修改全局生效，后续升级版本只需改此文件
