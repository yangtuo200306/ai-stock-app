## 3. 已实现功能

### 列表页重构

**布局结构：**
- 标题行："记录" + `(N)` 数量，与自选页风格一致
- 搜索框：实时过滤，匹配代码/名称/标题/摘要
- Pull-to-Refresh：替代旧版空态中的"刷新"按钮

**RecordCard 组件：**
- 左侧类型色条（4px 竖条，ask=蓝、report=青、analysis=紫）
- 紧凑 Badge + 时间（右对齐）
- 股票名称 + 等宽代码
- 标题（1 行）+ 摘要（2 行）
- 底部 meta 行：评分 + 涨跌幅，圆点分隔

**空态区分：**
- 搜索无结果："未找到匹配的记录"
- 真正无记录："暂无记录"

### 详情页视觉升级

**类型色条：**
- 顶部 4px 色条 + 紧凑 Badge，与 RecordCard 统一

**等宽代码：**
- `stock_code` 使用 `fontFamily: 'monospace'`

**分区标题：**
- 消息区："对话"小标题
- 指标区："行情摘要"小标题
- 视觉层级更清晰

**紧凑按钮：**
- "继续追问"和"查看完整报告"改为 Pressable + 主题色背景
- 替代旧版全宽 AppButton

**旧数据兼容：**
- 同时支持新版 `messages` 数组和旧版 `question`/`answer` 字段
