## 3. 已实现功能

### 布局重构
- 去掉旧版大 Card Header，改为紧凑标题行："自选股" + `(N)` 数量 + ⊕ 添加按钮
- 移除说明文案，页面信息密度更合理

### 搜索过滤
- 搜索框实时过滤已有自选股（匹配代码或名称，大小写不敏感）
- 自定义清除按钮（×），兼容 Android
- 搜索无结果时显示独立空态提示

### 可折叠添加表单
- 默认收起，点击 ⊕ 展开/收起
- 代码输入即时格式校验（normalizeAShareCode 结果反馈，边框变绿/变红 + ✓/✗）
- 添加成功后自动收起表单

### Pull-to-Refresh
- 使用 FlatList 原生 `refreshing` + `onRefresh`
- 下拉时并行请求 `loadStocks` + `loadTaskStatuses`
- 移除旧版"刷新列表"按钮

### 删除确认弹窗
- 点击删除图标弹出 ConfirmDialog
- 文案："确定删除 {stockName}({stockCode})？"
- 确认后执行删除，取消关闭弹窗

### 股票卡片升级
- 左侧类型色条（4px 竖条，基于 record_type 着色）
- 代码等宽字体 + 名称分行展示
- 紧凑圆角 Badge 替代旧版纯色标签
- 底部图标按钮（问 AI + 删除）替代全宽文字按钮

### 空态/错误态美化
- 空态：区分"搜索无结果"和"真正无自选股"
- 使用 StateView 组件统一展示
