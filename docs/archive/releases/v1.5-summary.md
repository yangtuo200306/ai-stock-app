## 3. 已实现功能

### AppButton compact 变体
- 新增可选 prop `size: 'default' | 'compact'`
- compact 尺寸：`minHeight: 36`，`borderRadius: 10`，`paddingVertical: spacing.sm`
- default 尺寸保持不变（`minHeight: 42`，`borderRadius: 12`）
- 所有旧页面不受影响（默认 `size='default'`）

### StateView 紧凑按钮
- 空态/错误态的操作按钮使用 `size="compact"`
- 视觉上更协调，与页面风格一致

### RecordCard 颜色统一
- 摘要文字颜色从 `colors.textMuted` 改为 `colors.textSecondary`
- 与 WatchlistStockCard 的摘要颜色一致
