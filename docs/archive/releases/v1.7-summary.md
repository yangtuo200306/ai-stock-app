## 遇到的问题

| 问题 | 原因 | 解决 |
|------|------|------|
| fastapi 0.135.1 装不了 | 需要 Python 3.10+ | pip 自动选 0.128.8 |
| uvicorn 0.41.0 装不了 | 需要 Python 3.10+ | pip 自动选 0.39.0 |
| pandas 编译慢 | 无 3.9 预编译包 | 装 `pandas<2.0` |
| numpy 不兼容 | numpy 2.x 不兼容 pandas 1.x | 装 `numpy<2` |
| `str \| None` 语法报错 | Python 3.10+ 语法 | 装 `eval-type-backport` |
| 端口占用 | 旧进程未退出 | `systemctl stop` 再 `start` |

## 经验

1. **服务器系统版本决定一切** — Ubuntu 20.04 的 Python 3.9 导致大量兼容问题，新项目建议用 22.04+
2. **个人项目用 `>=` 而非 `==`** — 本地和服务器环境不同时，固定版本反而装不上
3. **仓库公开简化部署** — HTTPS 克隆无需认证，后续 `git pull` 更新方便
